# Как затестить без бэкенда:

Установить json-server
```shell
npm install -g json-server
```

Перейти в каталог с проектом и ввести следующую команду.
```shell
npm run server
```

# Npm
#### Установить зависимости
```shell
npm run i
```

#### Development
```shell
npm run dev
```

#### Production
```shell
npm run prod
```

You'll find production files in ``prod/`` folder.
